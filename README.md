# Basic HTML Project – Game character,Sylus (Love and Deepspace)

This is a basic static webpage built using only HTML, A character called Sylus from Love and Deepspace.

## About the Project
- Created as part of my early web development practice  
- Focused on understanding core HTML structure  
- Includes simple styling using inline CSS (colors and layout)

## Features
- Basic webpage structure using HTML  
- Images and video integration  
- Simple form (review + email input)  
- Dark-themed layout  

## Tech Used
- HTML (no frameworks, no external CSS or JS)

## How to Run
Download or clone the repository and open `index.html` in a browser.

## Note
This is a beginner-level project focused on learning fundamentals.
